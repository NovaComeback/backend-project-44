# Brain Games

## Description
A collection of five text-based mini-games designed to train your logical thinking. The project is built as part of the Hexlet frontend/backend development curriculum.

### Games included:
* **Brain-Games:** Greeting module.
* **Brain-Even:** Check for even/odd numbers.

## Installation

Before installing, make sure you have Node.js version 20+ installed.

```bash
git clone <https://github.com/NovaComeback/backend-project-44>
cd frontend-project-44
